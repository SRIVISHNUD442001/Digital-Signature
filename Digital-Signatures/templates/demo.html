Demonstration



This is a live demo of how a digital signature is made and verified. A user can enter the bit size, if not it automatically is set to 1024, as well as some data, being some sort of text and then the data that would be received. If the data is the same, then the signature is verified and if not, then it is not verified. Once the data is entered and the submit button is pressed; the keys and signatures will show up, along with whether the signature is verified or not.
